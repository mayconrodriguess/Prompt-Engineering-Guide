# Guia de Prompt Engineering em Português-BR

Este repositório organiza conteúdos de prompt engineering em português-BR para estudo, consulta rápida e aplicação prática no dia a dia com IA.

## Objetivo

Reunir conceitos, técnicas, exemplos e referências que ajudem a escrever prompts melhores para LLMs, agentes e fluxos assistidos por IA.

## O que você vai encontrar aqui

- introdução a prompt engineering
- fundamentos para criação de prompts
- técnicas de prompting
- aplicações práticas em T.I. e automação
- links de estudo e referências úteis

## Estrutura sugerida

- [introducao/](./introducao/)
  Conceitos básicos, boas práticas e fundamentos.

- [tecnicas/](./tecnicas/)
  Zero-shot, few-shot, chain-of-thought, ReAct, RAG e outras abordagens.

- [aplicacoes/](./aplicacoes/)
  Casos de uso práticos para análise, documentação, automação, troubleshooting e produtividade.

- [modelos/](./modelos/)
  Observações sobre tipos de modelos, contexto, limitações e escolha de abordagem.

- [prompts/](./prompts/)
  Exemplos prontos e adaptáveis para diferentes tarefas.

- [referencias/](./referencias/)
  Links, artigos, materiais de apoio e fontes externas.

## Temas que serão trabalhados

- prompts para documentação técnica
- prompts para troubleshooting
- prompts para análise de logs e incidentes
- prompts para automação assistida
- prompts para geração e revisão de conteúdo
- prompting para agentes e fluxos multi-etapas
