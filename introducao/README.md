# Introdução

Esta pasta reúne os fundamentos de prompt engineering em português-BR.

## O que deve entrar aqui

- conceitos básicos
- boas práticas
- componentes de um prompt
- contexto, instruções e restrições
- erros comuns ao criar prompts

## Objetivo

Construir uma base sólida para entender como estruturar prompts melhores para LLMs, agentes e fluxos assistidos por IA.
